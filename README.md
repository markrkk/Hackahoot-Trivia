# Hackahoot Trivia 🏆

Juego de preguntas interactivo desarrollado con JavaScript.

## Descripción

Hackahoot Trivia es una pequeña aplicación web donde el usuario debe responder preguntas de desarrollo web seleccionando una de las opciones disponibles.

El juego comprueba automáticamente si la respuesta es correcta o incorrecta, actualiza el puntaje y pasa a la siguiente pregunta.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript

## Funcionalidades

- Mostrar preguntas dinámicamente
- Generar botones de respuestas automáticamente
- Verificar respuestas correctas e incorrectas
- Sistema de puntuación
- Cambio automático de preguntas
- Pantalla final al terminar el juego
- Diseño responsive básico
